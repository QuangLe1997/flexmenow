# FlexMe - Project Overview

## App Name
**FlexMe** (Domain: flexmenow.com)

## Tagline
> "Flex cuoc doi trong mo - chi voi 1 tam anh"

## Mo ta
FlexMe la web app (ho tro mobile + desktop) cho phep nguoi dung tao noi dung "flex" ban than bang AI.
Target: Gioi tre (Gen Z) muon the hien ban than tren mang xa hoi.

## 2 Tinh Nang Chinh

### Feature 1: Flex Anh (Single Image Generation)
- User upload anh chan dung
- Chon template co san (Luxury Car, Paris, Beach, CEO Office, ...)
- AI gen ra 1 anh chat luong cao voi mat cua user trong template
- Download / Share truc tiep len MXH
- Cost: 1 credit

### Feature 2: Flex Story (Story Series Generation)
- User upload anh chan dung
- Chon story pack:
  - "Du lich Paris 7 ngay" (10 anh)
  - "Co ny roi ne" (8 anh)
  - "CEO lifestyle" (6 anh)
  - "Gym transformation" (8 anh)
  - Custom stories (upcoming)
- AI gen ra chuoi anh lien tuc, co storyline
- User co the review, edit tung anh
- Schedule post len MXH (optional)
- Cost: 5-10 credits

## Dinh Vi San Pham
- **Entertainment / Creative Content Tool** - KHONG PHAI tool lua dao
- Khuyen khich user tag #FlexMe -> tao trend
- Watermark nhe "Made with FlexMe" -> branding tu nhien

## Target Users
- Gen Z (16-28 tuoi)
- Nguoi dung MXH active (Instagram, TikTok, Facebook, Threads)
- Thi truong: Viet Nam (phase 1), SEA (phase 2), Global (phase 3)
