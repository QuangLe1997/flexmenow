## 5. Ke Hoach Trien Khai

### Giai Doan 1: MVP (Thang 1-2)
**Muc tieu: Ra mat tinh nang Flex Anh**

| Tuan | Viec can lam | Ket qua |
|------|-------------|---------|
| Tuan 1-2 | Dung nen tang: website, dang nhap Google, database | Co trang web co the dang nhap |
| Tuan 3-4 | Tinh nang Flex Anh: upload anh, chon template, AI gen | User co the tao anh |
| Tuan 5-6 | Thanh toan, credit, gallery, fix bug | App hoan chinh, san sang ra mat |

**Khi xong se co:**
- Website chay tren dien thoai + may tinh
- Dang nhap bang Google
- 10-20 template (du lich, xe sang, lifestyle)
- Tao anh trong 15-30 giay
- Mua credits bang the quoc te

## 6. Cong Nghe Su Dung

| Thanh phan | Cong nghe | Giai thich don gian |
|-----------|-----------|-------------------|
| Website | Next.js | Framework lam web hien dai, nhanh |
| Giao dien | TailwindCSS | Lam giao dien dep, responsive |
| Dang nhap | Firebase Auth | Google, Apple login - an toan |
| Luu du lieu | Cloud Firestore | Database cua Google, nhanh, realtime |
| Luu anh | Firebase Storage | Kho luu anh tren Google Cloud |
| Backend | Cloud Functions | Code chay tren server cua Google |
| AI thong minh | Gemini AI | AI cua Google - xu ly logic, tao caption |
| AI tao anh | Vertex AI Imagen | AI cua Google - tao anh chat luong cao |
| Hosting | Firebase Hosting | Website chay tren mang luoi Google toan cau |
| Thanh toan | Stripe | Thanh toan the quoc te |

**Toan bo chay tren Google Cloud Platform - on dinh, bao mat, tu dong mo rong.**
